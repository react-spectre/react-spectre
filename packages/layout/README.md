# `@react-spectre/layout`
React components for Spectre.css's flexbox grid system

<br><br><br>
</div>

## Installation

```shell
yarn add @react-spectre/layout
```

```shell
npm install @react-spectre/layout --save
```

<br><br><br>

## Usage

### Grid

```js
import { Grid, Row, Col } from '@react-spectre/layout'
```

```jsx
    <Grid>
        <Row>
            <Col md={6}> md-6 </Col>
            <Col md={6}> md-7 </Col>
        </Row>
    </Grid>
```
